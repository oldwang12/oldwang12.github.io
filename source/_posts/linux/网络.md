#### 查看网络连接信息
```
netstat -nplt
```
-n 将字母转化为数字

-p 显示进程相关信息

-l 列出状态为监听

-t 只查看tcp协议

-a 查看全部协议(netstat -an)

#### 路由追踪
```
traceroute 8.8.8.8
```

#### 查看路由表
```
ip rule
ip -6 rule 
ip rule list
```
#### 查看默认路由表信息
```
ip r
ip -6 r
route
```
